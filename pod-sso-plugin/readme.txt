افزونه ورود یکپارچه پاد(POD SSO)

== Description ==
افزونه ورود یکپارچه پاد این امکان را به شما می دهد تا بتوانید ورود و احراز هویت پاد را در سایت وردپرسی خود داشته باشید.
* **برای استفاده از این افزونه شما باید حتما حساب کاربری کسب و کار پاد داشته باشید.**

= مستندات =
برای اطلاع از نحوه نصب و تنظیمات افزونه می توایند به [راهنمای نصب افزونه ورود یکپارچه پاد) ] (http://docs.pod.land/vv.1.0.0.0/PODSDKs/wordpressSDKs/3086/sso-plugin)
مراجعه نمایید

= امکانات =
* سیستم ورود یکپارچه
* پیاده سازی راحت سیستم احراز هویت
* عدم نیاز کاربر سامانه پاد به ثبت نام مجدد


== Installation ==
انتخاب افرودن افزونه جدید و جستجوی "افزونه ورود یکپارچه پاد"  یا دانلود افزونه و unzip کردن فایل.

= Minimum Requirements =

* WordPress 4.9 or greater
* POD SSO Plugin
