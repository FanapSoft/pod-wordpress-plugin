افزونه پرداخت امن پاد (پیپاد) با امکان پرداخت از طریق  درگاه بانک پاسارگاد و کیف پول پاد

== توضیحات ==
افزونه پرداخت پیپاد این امکان را به شما می دهد تا به راحتی از امکانات کیف پول و یا پرداخت مستقیم درگاه بانک پاسارگاد استفاده نمایید.
* **این افزونه به تنهایی قابل استفاده نمی باشد و نیاز به افزونه ورود یکپارچه پاد می باشد.**
* **برای استفاده از این افزونه شما باید حتما حساب کاربری کسب و کار پاد داشته باشید.**


= مستندات =
برای اطلاع از نحوه نصب و تنظیمات افزونه می توایند به [راهنمای نصب افزونه پیپاد) ] (http://docs.pod.land/vv.1.0.0.0/PODSDKs/wordpressSDKs/3131/paypod-plugin)
مراجع نمایید

= امکانات =
* امکان پرداخت مستقیم از طریق درگاه بانک پاسارگاد
* امکان پرداخت از کیف پول پاد
* ثبت فاکتور ها در پنل کسب و کار پاد شما
* امکان گزارش گیری پرداخت ها در پنل پاد


== Installation ==
انتخاب افرودن افزونه جدید و جستجوی "افزونه پرداخت امن پیپاد"  یا دانلود افزونه و unzip کردن فایل.

= Minimum Requirements =

* WordPress 4.9 or greater
* POD SSO Plugin
* WooCommerce
